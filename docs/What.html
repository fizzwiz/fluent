Class: What

What()

The What class provides a functional abstraction layer for fluent, composable logic.

Integrates seamlessly with Each and AsyncEach for declarative pipelines. Each instance is:

  • Callable like a function
  • Fluent via chainable methods

Shared semantic API (9 core methods): if(), sthen(), else(), which(), when(), match(), each(), self(), and what().

Constructor

new What()

Example
const doublePlusOne = What.as(x => x * 2)
  .if(x => x > 0)
  .sthen(x => x + 1);

doublePlusOne(5);       // => 11
doublePlusOne.what(5);  // => 11

else(f, errStringOrRegexopt) → {What}

Provide a fallback if this What/function returns undefined or throws a matching error.

This dynamic version of else allows:

  • Passing a fallback function or What instance f that will be evaluated if the primary function returns undefined.
  • Optionally filtering caught errors by a string or regular expression. Only errors whose message matches errStringOrRegex (or whose stringified form matches) will be swallowed; other errors are re-thrown.
Parameters:
Name Type Attributes Description
f function | What

Fallback function or What instance to use if the primary returns undefined.
errStringOrRegex string | RegExp <optional>

Optional error message string or RegExp to catch and use fallback. If omitted, any error is re-thrown.
Returns:

A new What instance that applies the fallback logic.

Type
What
Example
const fn = What.as(x => {
  if (x < 0) throw new Error("negative");
  return undefined;
}).else(x => x * 2, /negative/);

fn(-1); // => -2  (error "negative" matched, fallback applied)
fn(3);  // => 6   (primary returned undefined, fallback applied)
